====
IVX7
====

IVX7 is a tool for reading, modifying, and writing Yaesu VX-7[R] configurations.

---------
Reference
---------

I used `this website
<http://hse.dyndns.org/hiroto/RFY_LAB/vx7/e/vx7_8500.htm>`_ for all
reverse engineering information.

-----------
Buildsystem
-----------

Installation
++++++++++++

make INSTALL=1 ivx7.release
